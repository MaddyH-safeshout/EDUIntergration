Step 2: Configure Authentication

Set up OAuth authentication to allow SafeShout to access your Google Classroom data.

Required Steps:
  1. In the Google Cloud Console, navigate to APIs & Services > Credentials.
  2. Click Create Credentials and select OAuth client ID.
  3. Set the application type to Web application.
  4. Add your SafeShout domain as an authorized JavaScript origin.
  5. Add your callback URL as an authorized redirect URI.
  6. Copy the generated Client ID and Client Secret and paste them into the SafeShout admin panel.
